# Формы

## Press center (News) pages

### Запрос

POST: `/api/form/create-news`

Тип: `application/json`

Cтруктура данных

```
{
  email: "example@mail.ru"
}
```

### Ответ

Тип: `application/json`

Успешно подписался:

```
{
  "data": {
    status: "success"
  },
  "status": 200,
}
```

Уже подписан:

```
{
  data: {
    status: "subscribed"
  },
  status: 200
}
```

Ошибка:

```
{
  "data": {
    status: "{КОД ОШИБКИ}"
  },
  "status": 200,
}
```
